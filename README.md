Function to validate IMEI.

## Installation and Usage

Install the function with `npm install isValidImei`

#### No ES6

```javascript
var isValidImei = require('isValidImei');

isValidImei('123456789012345'); // true
```

#### ES6

```javascript
import isValidImei from 'isValidImei';

isValidImei('123456789012345'); // true
```
